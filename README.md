# Amazon Product Reviews Sentiment Analysis

## 📌 Project Overview

This project performs **Sentiment Analysis** on Amazon product reviews using **Natural Language Processing (NLP)** and **Machine Learning**. The objective is to classify customer reviews into **Positive**, **Neutral**, or **Negative** sentiments based on review text.

The project demonstrates the complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), text cleaning, feature extraction using TF-IDF, model training, evaluation, and prediction.

---

## 📂 Dataset

**Dataset:** Amazon Product Reviews

- Total Reviews: 568,454
- Source: Kaggle
- Features Used:
  - Score
  - Summary
  - Text
